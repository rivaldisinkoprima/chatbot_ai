# Zodiak AI Chatbot ✨🌟

A beautiful, modern chatbot focused on astrology and zodiac insights. Built with a cosmic theme featuring golden gradients, starry backgrounds, and smooth animations.

## 🌈 Features

- **Zodiac-Focused AI**: Specialized in astrology, horoscopes, zodiac compatibility, and celestial insights
- **Modern Cosmic UI**: 
  - Golden gradient theme with starry animated background
  - Smooth typing indicator animations
  - Avatar-based messaging (user & bot)
  - Copy message functionality
  - Clear chat option
  - Responsive design for all devices
- **Enhanced UX**:
  - Enter to send, Shift+Enter for new line
  - Character counter
  - Welcome message
  - Online status indicator
  - Markdown support (bold, italic, lists, code)
  - Keyboard shortcuts visible

## 🚀 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/zodiak-ai-chatbot.git
cd zodiak-ai-chatbot
```

2. **Install dependencies**
```bash
npm install
```

3. **Configure environment**
Create a `.env` file in the root directory:
```
GEMINI_API_KEY=your_google_gemini_api_key_here
```

4. **Run the application**
```bash
npm start
# or
node server.js
```

5. **Open in browser**
Visit `http://localhost:3000` to start chatting with Zodiak AI!

## 🎨 Design System

### Color Palette
- **Primary Gold**: `#fbbf24` (Zodiak Golden)
- **Primary Hover**: `#f59e0b`
- **Background**: `#0a0a0a` (Deep Space)
- **Chat Background**: `#0f0f0f` (Nebula Dark)
- **User Messages**: Golden gradient with dark text
- **Bot Messages**: Dark background with light text (`#e0e0e0`)
- **Accents**: Soft whites and muted grays

### Typography
- **Font**: Inter (clean, modern, highly readable)
- **Sizes**: Optimized for comfortable reading on all devices
- **Weights**: Regular, medium, semi-bold for hierarchy

## 💫 How It Works

The AI is specifically tuned to provide zodiac and astrology-related insights:

## 🛠️ Technical Stack

- **Backend**: Node.js with Express
- **AI**: Google Gemini API (gemini-3-flash-preview)
- **Frontend**: Vanilla HTML/CSS/JavaScript
- **Styling**: CSS Variables for easy theming
- **Animations**: CSS keyframes for smooth interactions
